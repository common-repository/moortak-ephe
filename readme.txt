== Description ==
this plugin is used to add a new widget to your elementor editor to show your portfolios in an awesome style with hover effect. all the options like changing colors, gradient, cont of items, responsive item cont and etc will be configured manually.

= Fa =

== Installation ==
1. Upload `moortak-ephe.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Go to your elementor editor and check out the new widget added to general widgets
